This is a basic chat app using Node.js and sockets to be added to a bigger app 
## Prerequisites
* Node.js
* [Yarn](https://yarnpkg.com/en/)

## Installing

Installing the dependencies

```bash
yarn
```

## Start the dev server

```bash
yarn start
```
